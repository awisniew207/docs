## Vincent Policies Created by the Vincent Team

| Policy                                       | Category          | Description                                                                                                                                            |
| -------------------------------------------- | ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [Contract Whitelist](./contract-whitelist) | Access Control    | Enforces strict access control for blockchain transactions by ensuring Vincent Apps can only interact with pre-approved smart contracts and functions. |
| [Spending Limit](./spending-limit)         | Financial Control | Enables Vincent App Users to set daily USD spending limits that restrict how much value Vincent Apps can spend on their behalf through transactions.   |
