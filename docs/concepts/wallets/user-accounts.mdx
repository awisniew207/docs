---
title: 'User Accounts'
---

Vincent User Accounts serve as the master identity for all your Vincent Wallets. When you authenticate with Vincent using any method, you create a single User Account that owns and manages all your Agent Wallets.

## Key Features

- **Centralized Management:** Provide centralized control for distributed wallet infrastructure
- **Multi-Auth Support:** Authenticate using existing web3 wallets, phone, email, or passkey
- **Hierarchical Control:** User Account owns and manages multiple Agent Wallets
- **Cross-Platform Access:** Access your wallets from any device or platform
